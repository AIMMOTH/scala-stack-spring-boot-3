Scala Stack

This is an example implementation of Scala Stack which uses Scala to create HTML, CSS and JavaScript on request. This means:

  • You can be 100% dynamic with all the resources in the response. You can create a custom HTML, CSS and JavaScript response.
  • The code is isomorphic (it uses a single language) which makes it simple to develop.
  • If you run it in development mode you can update most code and reload your page to instantly see your changes
  • No build or static files since everything is generated on a request

Scala Stack JS Compiler

This is a Spring Boot 3 Maven project setup that uses the external library Scala Stack JS Compiler to compile ScalaJS source to JavaScript. It works in the following process

  1. On Spring Boot startup it loads the necessary libraries for compilation of the source. It takes a few seconds and it is done once at startup to cache it.
  2. When a request for a JavaScript is made it loads all ScalaJS files which are available at server runtime and compiles them to a JavaScript.
  3. To speed things up you can cache the result
